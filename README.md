## Lab 01 – VLAN Creation and Access Port Configuration

### Objective

Configure a Virtual Local Area Network (VLAN), assign switch access ports, configure IP addresses for end devices and verify successful communication within the VLAN.

### Implementation

- Created a new VLAN and assigned an appropriate VLAN name.
- Configured a Switch Virtual Interface (SVI).
- Assigned switch access ports to the VLAN.
- Connected two PCs using UTP cables.
- Configured IPv4 addresses.
- Verified communication using the **ping** command.

### Skills Developed

- VLAN Configuration
- Access Port Configuration
- IPv4 Addressing
- Switch Configuration
